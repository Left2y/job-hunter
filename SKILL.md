# JobHunter — 求职全流程助手

## 🎯 核心功能

1. **岗位搜索** - BOSS 直聘 MCP 自动抓取
2. **自动打招呼** - 个性化招呼语批量发送
3. **JD 解析** - 与简历匹配度分析
4. **公司背调** - 风险信号检测（劳动仲裁/欠薪/证监会警示）
5. **面试准备** - 公司资料 + 问题预测
6. **面试策略** - 谈薪建议 + 风险评估
7. **跟进管理** - SQLite 存储 + 进度追踪

---

## 🚀 快速开始

### 前置要求

1. **BOSS 直聘 MCP** - 已配置并运行
2. **Agent-Reach** - 已安装
3. **mcporter** - 已安装

### 检查配置

```bash
# 1. 检查 BOSS MCP 状态
mcporter list | grep bosszp

# 2. 测试连接
mcporter call 'bosszp.get_recommend_jobs(page: 1)'

# 3. 如未运行，启动 BOSS MCP Server
# cd /path/to/bosszp-mcp && node server.js
```

## 🔧 技术架构

### 核心依赖

| 工具 | 用途 |
|------|------|
| **BOSS MCP** | 岗位搜索/打招呼/聊天 |
| **ask-search** | 公司背调 |
| **Tavily API** | 新闻/舆情搜索 |
| **browser 工具** | 看准网/脉脉评价 |
| **Qwen/Gemini** | 匹配度分析 |
| **SQLite** | 数据存储 |

### 文件结构

```
~/.openclaw/skills/job-hunter/
├── SKILL.md              # 本文件
├── src/
│   ├── boss-search.js    # BOSS 岗位搜索
│   ├── auto-greeting.js  # 自动打招呼
│   ├── jd-analyzer.js    # JD 解析
│   ├── company-check.js  # 公司背调
│   ├── interview-prep.js # 面试准备
│   └── tracker.js        # 进度管理
├── data/
│   └── applications.db   # SQLite
└── templates/
    └── greeting.md       # 招呼语模板
```
